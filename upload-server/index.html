Pocket Fische Upload Server

Welcome to the CW&T Pocket Fische parcel upload system.

What is this?

This system allows Kickstarter backers to upload their custom images to be included on the Pocket Fische microfiche.

I'm a backer with an upload link

If you received an upload link from the project admin, click that link to upload your image.

Your link will look like: upload.html?code=XXXXXXXX

I'm a project admin

If you're an admin, you should have a bookmark to the admin page with your admin-id.

The admin page allows you to generate upload codes for backers.

View the world

See all uploaded parcels on the Pocket Fische viewer
